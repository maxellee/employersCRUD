Підйом даних з нижніх компонентів в ієрархії називаєтться Підйомом стану(state lifting)

Доступність контенту - це спеціальні технічні і архітектурні рішення, які допомагають людям з обмеженими можливостями
використовувати сайт. Абревіатура доступності контенту a11y
WCAG - web content accessibility guidelines

Проблема інлайн стилів в реакті це те що він автоматично не доставляє вендорні префікси. Це доводиться робити самому вручну
Для передачі стилів можна використовувати об'єкт раніше об'явлений або створити одразу в атрибуті style={{}}.
Реакт автоматично підставляє всі значення в px. Для зміни значення необхідно число взяти в лапки та додати вимірювальну систему

При роботі зі стилями від препроцессорів, вкладені файли не наслідують вниз по ієрархії змінні, тому їх доводиться імпортувати вручну
Для коректної роботи препроцессора sass та scss необхідно встановити npm пакет
npm i sass --savedev
В даному випадку бейбел зберігатиме файли scss в окремих тегах style. Це відбувається через режим розробки
